# 陽明交大電競社Discord架構手冊

## 身分組

### 1. 分類身分組
- 作用
  1. 人員分流
  2. 可自行關閉不想收聽的遊戲類別
- 當前
  - ```傳說對決-AOV```
  - ```英雄聯盟-LOL```
  - ```特戰英豪-Valorant```
  - ```Apex Legends```
  - ```綜合遊戲```

### 2. 角色身分組
- 作用
  1. 方便@對應人員
  2. 同角色統一管理
- 當前
  - 電競社
    - ```社員```
    - ```幹部```
      - 擁有```社員```的所有權限
      - 另外擁有建立特定頻道訊息的能力
    - ```網管```
      - 擁有```幹部```的所有權限
      - 另外有Discord以及機器人管理的權限
  - 非電競社的人類帳號
    - ```待定(請自我介紹```
    - ```Guest```
  - 機器人
    - ```NYCU_Esport_Bot```

### 3. 比賽身分組
- 作用
  1. 根據每次比賽建立
  2. 持有身分組的為選手、裁判
  3. 統一發布比賽進程
- 當前
  - ```2023-梅竹校內選拔賽```

## 入群流程
1. 點選Discord永久連結
  - ```https://discord.com/invite/uX3PqeyWE2```
2. Discord的"培訓"功能設定詢問兩個問題
  1. 選擇您想收聽的遊戲種類
     - 選項是當前所有遊戲身分組 [說明](https://github.com/NYCU-Esport/Discord-Docs/blob/main/Architecture.md#1-%E5%88%86%E9%A1%9E%E8%BA%AB%E5%88%86%E7%B5%84)
     - 選哪個就會自動賦予那個身分組
  3. 您是否是交大電競社社員?
     - 是
       1. 賦予```待定(請自我介紹```身分組
       2. 管理員審核以交社費後更改暱稱並修身分組成```社員```or ```幹部```
     - 否
       - 賦予```Guest```身分組
       - 如果```Guest```身分組是比賽選手
         1. 經過管理員審核
         2. 移除```Guest```身分組
         3. 新增比賽身分組，如```2023-梅竹校內選拔賽```
         4. 比賽身分組擁有```Guest```身分組所有權限

## 頻道權限
- 簡寫
  - n : 不可檢視或發言 (not allow)
  - r : 可檢視 (read)
  - w : 可發言 (write)
- 資訊 Information
  1. 使用規範
    - 作用
      1. 進來的成員第一眼就會看到這
      2. 只放超級重要的東西 (不要罵人等等這種)
    - 權限
      - ```所有人```: r
      - ```管理員```: r+w
  2. 系統訊息
    - 作用
      1. 系統自動推送訊息
      2. 系統會自動產生
         - 成員進入
         - Discord更新通知
    - 權限
      - ```所有人```: r
  3. 社團公告
    - 作用
      1. 幹部宣布的重要事情
      2. 盡量精簡、條列式
    - 權限
      - ```所有人```: r
      - ```幹部```: r+w
  4. 競賽資訊
    - 作用
      1. 公告競賽資訊並附報名連結表單
      2. 盡量精簡、條列式
    - 權限
      - ```所有人```: r
      - ```幹部```: r+w
  5. 公開資源
    - 作用
      1. 儲存社團相關資訊
      2. 盡量精簡、條列式
    - 權限
      - ```所有人```: n
      - ```Guset```: r
      - ```社員```: r+w
  6. 貼文中心
    - 作用
      1. 貼文集中串 (像是二手版)
    - 權限
      - ```所有人```: n
      - ```Guset```: r+w
      - ```社員```: r+w
